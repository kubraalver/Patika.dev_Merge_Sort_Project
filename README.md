# Merge Sort Project

---

## Merge Sort Steps
1. [16,21,11,8,12,22]

2. [16,21,11] - [8,12,22]

3. [16,21][11] - [8,12][22]

4. [16] [21] [11] - [8] [12] [22]

5. [11,16,21] - [8,12,22]

6. [8,11,12,16,21,22]

---

## Big O Notation
2^x= n, logn= x,

Worst Case: O(nlogn)

Average Case: O(nlogn)

Best Case: O(nlogn)
